2. 下载下来后是未注册的状态，需要破解一下。破解方法：     
(1) 打开网址  https://hexed.it/    
(2) 打开sublime_text.exe文件    
(3)替换:
            `80 78 05 00 0F 94 C1`
      -->
            `C6 40 05 01 48 85 C9`    
(4)保存。    
(5)打开软件，点击帮助-关于查看破解效果
****

(2)菜单点击Preferences – Package Control，选择 Install Package
